音声でメモを簡単に管理できるAlexaスキル

✅ Production Ready
Version 1.0.0
📝

メモ追加

「牛乳を買うをメモして」と話すだけで、簡単にメモを追加できます。

📢

メモ読み上げ

「メモを読んで」と言えば、保存されているメモを順番に読み上げます。

🗑️

メモ削除

「1番目のメモを削除」のように番号を指定して削除できます。

使い方

1. 「アレクサ、ボイスメモを開いて」

2. 「牛乳を買うをメモして」

3. 「メモを読んで」
